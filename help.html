block sites help! information


Question: Do I have to press save? (on the settings page)

Once you press save then all the text you wrote in the inputs and all the buttons that you pressed will be saved to your chrome local storage. It only saves with the save button so I don't have to keep writing stuff to storage constantly. (more efficient and easier)


Question: Why are there different groups? How should I distinguish my groups?

Different groups should have different sites that are at different times. For example, if I wanted to block twitter in the morning and youtube at night, I would use two different groups. If I wanted to block twitter and youtube in the morning and at night, I would use one group. If I wanted to block youtube in the morning and at night, but twitter only in the morning, I would use two groups.


Question: What does the "block all sites" button do?

This button accesses all the websites from every group that you have active (selected green for on) and blocks those websites from midnight until midnight.


Question: How do I delete a site or time input field on the settings page?

You cannot. If the input field for a site is empty (no text written), then it will not be saved to storage and when you next visit the settings page it will not be there. If the start time is the same as the end time, it will similarly not be recorded or displayed.


Question: I made a change on settings, pressed the save button, reloaded, and the page hasn't changed?

Once you click the save button, all your new settings are collected and given to Chrome storage. This can sometimes be slow, depending what else you're doing on Chrome. It will be written, but not instantly. Whenever you reload the settings page, it gets whatever data is already in Chrome's storage, which can be not up to date.


Question: Can I change / add to the photos that show on the blocked site?

YES! First, move your image into the folder titled "images" within the "blockSites" extension folder. Then, put the name of those images and some alt text in the file "blocked.js". There are more instructions of where exactly to write those within that file.


Question: I put the same website in sites to block and sites to exclude from blocking?

It won't block. If, in the same time period, you have "youtube.com" in the block list but also you have "youtube.com" in the exclude list, it will not block. What sites block what depends on being in the same time range, not the same group.
